# 프로젝트 설명

음원 스트리밍 웹사이트

# 기술 스택

# 프로그램 기능

- 회원가입, 로그인, 로그아웃 기능 제공
- 글 작성, 수정, 삭제, 열람이 가능한 게시판 기능 제공
- 시대별, 장르별 카테고리로 음악 제공
- 실시간 음원 스트리밍과 다운로드 제공
